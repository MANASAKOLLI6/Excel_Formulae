SUM: Adds all the numbers in a range of cells

Syntax: =SUM(number1,[number2],..)

Shortcut: Alt + =

SUMIF: Adds the cells specified by a given condition or criteria

Syntax: =SUMIF (range, criteria,[ sum_range])

SUMIFS: Adds the cells specified by a given set of conditions or criteria

Syntax: =SUMIFS(sum_range,criteria_range1,criteria1,..)

X

AVERAGE: Returns the average(Arithemetic mean)of its arguments

Syntax: =AVERAGE (number1, [number2], ...)

Shortcut: Alt + h + u + a

AVERAGEIF: Finds average for the cells specified by a given condition or criteria

Syntax: =AVERAGEIF (range, criteria,[average_range])

AVERAGEIFS: Finds average for the cells specified by a given set of conditions or criteria

Syntax: =AVERAGEIFS(average_range,criteria_range1,criteria1,..)

X

IF: Checks whether a condition is met, and returns one value if true and another value if false.

Syntax: =IF(logical_test, value_if_true, [value_if_false])

IFERROR: To trap and handle errors in a formula.
  • If an error exists - returns a value we specified
  • No error - returns the result of the formula

    • Syntax: =IFERROR(value, value_if_error)

    IFNA: Returns the value we specify if the expression resolves to #N/A, otherwise returns the result of the expression

    Syntax: =IFNA(value, value_if_na)

    X

    COUNT: Counts the no.of cells in a range that contain numbers

    Syntax: =COUNT(value1, [value2], ...)

    COUNTA: Counts the no.of cells in a range that are not empty

    Syntax: =COUNTA(value1, [value2], ...)

    COUNTIF: Counts the no.of cells within a range that meet the given condition

    Syntax: =COUNTIF(range,criteria)

    COUNTIFS: Counts the no.of cells specified by a given set of conditions

    Syntax: =COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2]…)

    X

    OR: returns True if at least one of the criteria evaluates to true. If all the arguments evaluate as False, then returns False

    Syntax: =OR(logical1, [logical2], …)

    AND: Check whether all arguments are true, and returns true if all arguments are true

    Syntax: =AND(logical1, [logical2]…)

    NOT: Changes false to true,or true to false

    Syntax: =NOT(logical)

    X

    ISTEXT: Checks whether a value is text, and returns true or false

    Syntax: =ISTEXT(value)

    ISBLANK: Checks whether a reference is to an empty cell, and returns true or false

    Syntax: =ISBLANK(value)

    X

    MAX: Returns the largest value in a set of values. Ignores logical values and text

    Syntax: =MAX(number1, [number2], ...)

    MIN: Returns the smallest value in a set of values. Ignores logical values and text

    Syntax: =MIN(number1, [number2], ...)

    X

    HLOOKUP: Can locate and retrieve a value from data in a horizontal table

    Syntax: =HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

    VLOOKUP: when you need to find things in a table or a range by row

    Syntax: =VLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

    =LOOKUP(What you want to look up, where you want to look for it, the column number in the range containing the value to return, return an Approximate or Exact match -indicated as 1/TRUE, or 0/FALSE).

    X

    Filter: Filter a range of data based on criteria we define

    Syntax: =FILTER(array, include, [if_empty])

    Shortcut: Ctrl + Shift + L

    X

    Index: Returns the value at a given location in a range or array.

    Syntax: =INDEX(array, row_num, [col_num], [area_num])

    X

    Match: Locate the position of a lookup value in a row, column, or table.

    Syntax: =MATCH(lookup_value, lookup_array, [match_type])

    X

    Row: Returns the row number of a reference

    Syntax: =ROW([reference])

    Column: Returns the column number of a reference

    Syntax: =COLUMN([reference])

    X

    True: Returns the logical value TRUE

    Syntax: =TRUE()

    False: Returns the logical value FALSE

    Syntax: =FALSE()

    X

    Trim: Removes the extra spaces from the start and end of the cell value.

    Syntax: =TRIM(text)

    Clean: Removes all nonprintable characters from text

    Syntax: =CLEAN(text)

    X

    Left: Extracts a given number of characters from the left side of a supplied text string.

    Syntax: =LEFT(text,[num_chars])

    Right: Extracts a given number of characters from the left side of a supplied text string.

    Syntax: =RIGHT(text,[num_chars])

    X

    Find: Get location substring in a string(case sensitive)

    Syntax: =FIND(find_text, within_text, [start_num])

    Shortcut: Ctrl + F

    Search: Get the location of substring in a string(not case sensitive)

    Syntax: =SEARCH(find_text, within_text, [start_num])

    Shortcut: Alt + Q

    X

    EOMONTH: Get last day of month n months in future or past

    Syntax: =EOMONTH (start_date, months)

    X

    Concat: combines the text from multiple ranges and/or strings, but it doesn't provide delimiter or IgnoreEmpty arguments.

    Syntax: =CONCAT(text1, [text2],…)

    X

    TEXTJOIN: Join text values with a delimiter

    Syntax: =TEXTJOIN(delimiter, ignore_empty, text1, [text2], ...)

    TEXT: Convert a number to text in a number format

    Syntax: =TEXT(value, format_text)

    X

    MID: Extracts a given number of characters from the middle of a supplied text string.

    Syntax: =MID(text, start_num, num_chars)

    X

    EXACT: Shows whether both data are an exact match or not.

    Syntax: =EXACT (text1,text2)

    X

    NetworkDays: To get the number of business days between dates.

    Syntax: =NETWORKDAYS(start_date,end_date,holidays)
