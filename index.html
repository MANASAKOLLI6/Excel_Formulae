SUM : Adds all the numbers in a range of cells

Syntax : =SUM(number1,[number2],..)

Shortcut : Alt + =

SUMIF : Adds the cells specified by a given condition or criteria

Syntax : =SUMIF (range, criteria,[ sum_range])

SUMIFS : Adds the cells specified by a given set of conditions or criteria

Syntax : =SUMIFS(sum_range,criteria_range1,criteria1,..)

X

AVERAGE : Returns the average(Arithemetic mean)of its arguments

Syntax : =AVERAGE (number1, [number2], ...)

Shortcut : Alt + h + u + a

AVERAGEIF : Finds average for the cells specified by a given condition or criteria

Syntax : =AVERAGEIF (range, criteria,[average_range])

AVERAGEIFS : Finds average for the cells specified by a given set of conditions or criteria

Syntax : =AVERAGEIFS(average_range,criteria_range1,criteria1,..)

IF : Checks whether a condition is met, and returns one value if true and another value if false.

Syntax : =IF(logical_test, value_if_true, [value_if_false])

IFERROR : To trap and handle errors in a formula.
  • If an error exists - returns a value we specified
  • No error - returns the result of the formula

    • Syntax : =IFERROR(value, value_if_error)

    IFNA : Returns the value we specify if the expression resolves to #N/A, otherwise returns the result of the expression

    Syntax : =IFNA(value, value_if_na)

    COUNT : Counts the no.of cells in a range that contain numbers

    Syntax : =COUNT(value1, [value2], ...)

    COUNTA : Counts the no.of cells in a range that are not empty

    Syntax : =COUNTA(value1, [value2], ...)

    COUNTIF : Counts the no.of cells within a range that meet the given condition

    Syntax : =COUNTIF(range,criteria)

    COUNTIFS : Counts the no.of cells specified by a given set of conditions

    Syntax : =COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2]…)

    OR : returns True if at least one of the criteria evaluates to true. If all the arguments evaluate as False, then returns False

    Syntax : =OR(logical1, [logical2], …)

    AND : Check whether all arguments are true, and returns true if all arguments are true

    Syntax : =AND(logical1, [logical2]…)

    NOT : Changes false to true,or true to false

    Syntax : =NOT(logical)

    ISTEXT : Checks whether a value is text, and returns true or false

    ISBLANK : Checks whether a reference is to an empty cell, and returns true or false

    Table : To create a table to organize & analyze related data.

    Shortcut : Ctrl + T

    Process :

    1. Select a cell within your data.
    2. Select Home > Format as Table.
    3. Choose a style for your table.
    4. In the Create Table dialog box, set your cell range.
    5. Mark if your table has headers.
    6. Select OK.

    X

    Pivote Tbale : To quickly summarize large amounts of data.

    Process :

    1. Select the cells you want to create a PivotTable from.
    2. Select Insert and choose the top of the PivotTable button PivotTable button.
    3. Under Select a table or range, and verify the cell range in Table/Range.
    4. Select New worksheet or Existing worksheet under Choose where you want the PivotTable to be placed
    5. Select OK.

    X

    Charts : Enable you to visually compare multiple sets of data.

    Process :

    1. Select the data you'd like to include in the chart.
    2. On the Insert tab, select your desired chart in Recommended Charts.
    3. See charts with your data previewed in the Recommended Charts pane.
    4. Choose one and select Insert Chart.

    X

    Slicer : To filter data visually.

    Shortcut : Alt + N + S + F

    Process :

    1. Click anywhere in the table or PivotTable.
    2. On the Insert tab, select Slicer.
    3. In the Insert Slicers dialog box, select the check boxes for the fields you want to display, then select OK.
    4. A slicer will be created for every field that you selected. Clicking any of the slicer buttons will automatically apply that filter to the linked table or PivotTable.

    X

    Conditional Formatting : To make it easy to highlight certain values.

    Process :

    1. Select the cells you want to format.
    2. Go to Home and select Conditional Formatting.
    3. Select the desired options for your data.

    X

    Data Validation : To pick from a list of rules to limit the type of data that can be entered in a cell.

    Process :

    1. Select the cells that should have the drop-down, go to Data and select Data Validation.
    2. For Allow, choose List.
    3. For Source, type drop-down items separate by commas.
    4. For example: Yes,No,Maybe.

    X

    Duplicate Removal : To remove the duplicate values

    Process :

    1. Select the range of cells that has duplicate values you want to remove.
    2. Click Data > Remove Duplicates, and then Under Columns, check or uncheck the columns where you want to remove the duplicates.
    3. Click OK.

    X

    Freeze Panes :Locks rows, columns, or both to the top and left in a sheet.

    Process :

    1. Select the cell below the rows and to the right of the columns to freeze.
    2. Go to View, select Freeze panes button Freeze Panes or press Alt+W+F, and choose what to freeze:
      • Both rows & columns: Select Freeze Panes
      • Just the top row: Select Freeze Top Row.
      • Just the first column: Select Freeze First Column.

    Protecting sheet/workbook : To prevent other users from accidentally or deliberately changing, moving, or deleting data in a worksheet.

    Process :

    1. Go to File. Once you click on the Info, you will see the Protect Workbook option on the right side.
    2. Click on the drop-down list of Protect Workbook option and select Encrypt with Password.
    3. Enter the password according to your wish and Click OK. ...
    4. It will again ask you to re-enter the password.

    Grouping & Ungrouping rows/columns: To organize a comparable layout

    Process :

    1. Select the row or column you want to group.
    2. Click on “Group” on the Data ribbon.

      3. Shortcut - Group : Alt + shift + arrow right

      Shortcut - Ungroup : Alt + shift + arrow left
